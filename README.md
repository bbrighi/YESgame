YESgame
=======

YES game è una webapp che calcola la libertà finanziaria utilizzando lo schema del gioco Y€$ di Alfio Bardolla.
Si tratta di un calcolatore ad hoc, che sostituisce la scheda bilancio presente nel gioco.
